# BareWire

Welcome to the official documentation for **BareWire** — a high-performance async messaging library for .NET 10 / C# 14.

BareWire is an alternative to MassTransit with fundamentally different architecture: raw-first (no envelope by default), zero-copy pipeline, manual topology, and deterministic memory usage.
