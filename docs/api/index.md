# API Reference

The API reference is generated automatically from XML documentation comments in the source code.

After running `docfx metadata`, this folder will be populated with YAML files describing every public type in the configured projects. DocFX will render them into browsable HTML pages in the built site.

Use the left-hand navigation to explore namespaces, types, and members.
